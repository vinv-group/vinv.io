# Getting started

## .vinv Exchange Format ##

.vinv is a format for encoding individual tree data.

{% highlight javascript %}
{
    "LURPdZ8NixY7WriS22S01":[
        [
            [23.4444441,23.4464441,0],
            12
        ],
        "666", // dsff
        25, // dsff
        22,
        [
            [0.3,5],
            [0.6,5]],
            [[0.3,5],
            [0.6,5]
        ],
        "2018-11-13",
        null
    ]
}
{% endhighlight %}
